## Running the Backend (Blog Publishing API)

1. Make sure you have Python 3 installed.
2. Install Flask:
   ```sh
   pip install flask
   ```
3. Go to the `backend` directory:
   ```sh
   cd backend
   ```
4. Run the server:
   ```sh
   python app.py
   ```
5. The API will be available at `http://localhost:5000`.
